# Установка Canvibe

## Требования

1. **Node.js** (v18+) - уже установлен ✓
2. **Rust** - нужен для Tauri приложения

## Установка Rust

Для Windows:

1. Скачайте и установите [rustup-init.exe](https://rustup.rs/)
2. Или через PowerShell:
   ```powershell
   winget install Rustlang.Rustup
   ```
3. После установки перезапустите терминал
4. Проверьте установку:
   ```bash
   rustc --version
   cargo --version
   ```

## Запуск приложения

### Вариант 1: Веб-версия (для разработки без Rust)

```bash
npm run dev
```

Откроется на `http://localhost:1420`

**Внимание:** В веб-версии будут ограничения:
- Нет доступа к файловой системе
- Нет интеграции с Git
- Нет нативного доступа к системе

### Вариант 2: Десктопное приложение (нужен Rust)

После установки Rust:

```bash
npm run tauri:dev
```

Это запустит полноценное десктопное приложение с полным доступом к системе.

## Сборка для продакшна

```bash
npm run tauri:build
```

Собранные файлы будут в `src-tauri/target/release/`
