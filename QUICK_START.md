# Быстрый старт

## Проблема: Rust не установлен

Для запуска полноценного десктопного приложения нужен Rust.

## Решение: Запуск веб-версии

Сейчас запущена веб-версия приложения:

1. Откройте браузер
2. Перейдите по адресу: **http://localhost:1420**

## Что работает в веб-версии:

✅ Базовый UI и холст  
✅ Event-log система (использует localStorage)  
✅ Визуализация узлов  
✅ Failure Lens  
❌ Git интеграция (требует Tauri)  
❌ Мониторинг файловой системы (требует Tauri)  

## Установка Rust (для полной версии):

### Windows:
1. Скачайте: https://rustup.rs/
2. Или через PowerShell:
   ```powershell
   winget install Rustlang.Rustup
   ```
3. Перезапустите терминал
4. Проверьте: `rustc --version`
5. Запустите: `npm run tauri:dev`
