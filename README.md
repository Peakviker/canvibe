# Canvibe

Визуальный оркестратор мышления для разработки с AI.

## Концепция

Приложение визуализирует процесс разработки на бесконечном холсте:
- Планы и выполнение задач
- Структура сайтов и компонентов
- Ветки разработки
- AI-размышления (Cursor/GPT)
- Созданные файлы и коммиты

## Технологии

- **Tauri** - десктопное приложение
- **React + TypeScript** - UI
- **Konva** - canvas для визуализации
- **Event-log система** - NDJSON формат для хранения событий

## Разработка

```bash
npm install
npm run tauri:dev
```

## Сборка

```bash
npm run tauri:build
```
